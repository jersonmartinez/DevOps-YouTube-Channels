Criterios de Selección
  • Contenido de alta calidad relacionado con DevOps
  • Videos regulares y actualizados
  • Creador reconocido en la comunidad
  • Contenido en español o inglés
  • Enfoque educativo o técnico

Guía de Contribución

Criterios para Canales:
  • Contenido relevante: DevOps, Cloud, Platform Engineering, Containers, etc.
  • Calidad técnica: Información precisa y bien explicada
  • Regularidad: Uploads consistentes (al menos 1 video por mes)
  • Comunidad: Engagement activo con la audiencia
  • Experiencia: Creador con experiencia demostrable

Para desarrolladores:
# 1. Fork el repositorio
git clone https://github.com/jersonmartinez/DevOps-YouTube-Channels.git

# 2. Crea una rama para tu contribución
git checkout -b add-channel-nombre-canal

# 3. Agrega el canal en el archivo correspondiente
# Ejemplo en categories/cloud.md:

### Nuevo Canal
[![@NuevoCanal](https://img.shields.io/youtube/channel/subscribers/CHANNEL_ID?label=%40NuevoCanal&style=social)](https://www.youtube.com/@NuevoCanal?sub_confirmation=1)

**Canal**: https://www.youtube.com/@NuevoCanal
**LinkedIn**: [Autor](https://linkedin.com/in/autor)
**Rol**: DevOps Engineer
**Etiquetas**: `#aws` `#terraform` `#kubernetes`

#### 🎯 Contenido Destacado
- Descripción del contenido
- Temas principales
- Valor para la comunidad
